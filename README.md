# SnapsHire Project

A REST API backend for **SnapsHire**, a photographer booking platform built with **Python, Django, and Django REST Framework**.

Features

 User
- User Registration & Login (JWT Authentication)
- View Photographer Profiles
- Search & Filter Photographers
- Book Photographer
- Booking History
- Cancel Booking
- User Profile Management

Photographer
- Photographer Registration
- Profile Management
- Portfolio Upload
- Availability Management
- Booking Management
- Earnings Dashboard

Admin
- Manage Users
- Manage Photographers
- Verify Photographer Accounts
- Manage Bookings
- Manage Payments
- Manage Banners
- Dashboard Statistics

---

Tech Stack

- Python 3
- Django
- Django REST Framework
- MySQL
- JWT Authentication
- Swagger API Documentation
